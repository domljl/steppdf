# LibreOffice Best-Effort Conversion

StepPDF will use LibreOffice first for document-to-PDF conversion while aiming for output that is as close as practical to PowerPoint or Word export. This avoids external office automation services for the first version, but accepts that some files may render differently when fonts, animations, or advanced layout features are involved; Microsoft-based conversion remains the likely upgrade path if fidelity is not good enough.
